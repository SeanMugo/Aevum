# Aevum – Luxury Automotive Dealership

A premium, custom-built website for a luxury car dealership. Designed with a timeless aesthetic to reflect exclusivity, heritage, and elegance.

---

## 📖 Overview

Aevum (Latin for "age" or "eternity") is a multi-page website for a high-end collector car dealership. The site focuses on visual storytelling, minimal luxury design, and a seamless user experience.

---

## 🚗 Pages

| Page | Description |
|------|-------------|
| **Home** (`index.html`) | Hero section with "Timeless Machines" headline, brand philosophy, featured vehicles, concierge team, client testimonials, journal preview |
| **Collection** (`collection.html`) | Full inventory of all available vehicles with pricing, mileage, and condition details |
| **Curated** (`curated.html`) | Hand-picked, exceptional vehicles — the rarest and most significant cars |
| **Concierge** (`concierge.html`) | Meet the team of specialists: Principal Concierge, Acquisition Specialist, and Heritage Director |
| **Journal** (`journal.html`) | Articles on automotive heritage, driving experiences, and collector insights |
| **Inquire** (`inquire.html`) | Contact form and consultation booking |

---

## 🛠️ Built With

- **HTML5** – Structure
- **Tailwind CSS** – Styling and responsiveness
- **Font Awesome** – Icons
- **CSS3** – Custom animations and mobile menu toggle

## 🎨 Color Palette

| Color Name | Hex Code | Usage |
|------------|----------|-------|
| Eternal Black | `#0D0D0D` | Primary background |
| Marble White | `#F5F1E8` | Body text |
| Patina Gold | `#B5A37E` | Accents, hover states, active page |
| Midnight Bronze | `#4A3F33` | Borders and dividers |

---

## ✨ Features

- Fully responsive design (mobile, tablet, desktop)
- Sticky navigation bar with active page highlighting
- Mobile hamburger menu using pure CSS (checkbox hack)
- Smooth hover animations on cards and buttons
- Luxury-focused typography and generous spacing
- Consistent section dividers in brand colors
